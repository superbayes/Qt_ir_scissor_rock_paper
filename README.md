# Qt_ir_scissor_rock_paper
Rock Paper Scissors recognition based on infrared camera(基于红外相机的剪刀石头布识别)

## tech points(技术要点)
* Qt 5.12.9
* c++
* opencv 4.5.2
* nanodet
* ncnn
* USB相机(flir lepton 红外模组,分辨率160*120)
* windows10
